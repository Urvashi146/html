JavaScript Function bind()

This example creates 2 objects (b1 and b2).

The b2 object borrows the fullname method from b1:

    const b1 = {
        firstName:"urvashi",
        lastName: "barot",
        fullName: function() {
          return this.firstName + " " + this.lastName;
        }
      }
      
      const b2 = {
        firstName:"uri",
        lastName: "shah",
      }
      
      let fullName = b1.fullName.bind(b2);
      
      document.getElementById("bd").innerHTML = fullName();
